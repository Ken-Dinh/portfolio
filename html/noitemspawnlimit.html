NoItemSpawnLimit

8 Juin 2025

C# Unity BepInEx

Premier projet de modding sur un jeu Unity.
Découverte complète de l'écosystème des jeux Unity et du modding. Création d'un mod utile pour REPO permettant de supprimer ou personnaliser les limites de spawn d'objets, y compris ceux ajoutés par d'autres mods.

GitHub Thunderstore

Motivation & Inspiration

L'idée de créer ce mod est née d'un besoin personnel : créer un modpack pour jouer avec des amis. Nous avions rencontré un problème récurrent dans REPO - les limites de spawn des objets achetés dans le camion empêchaient certaines objets de spawn.

Après avoir cherché une solution parmi les mods existants, personne n'avait résolu ce problème spécifique. C'était l'occasion idéale pour créer mon premier mod Unity et découvrir l'écosystème du modding.

Pour faciliter l'installation du mod pour mes amis, j'ai décidé de le publier sur Thunderstore. Ce qui a commencé comme une solution personnelle a pris une dimension communautaire - le mod reçoit maintenant plus de 500K téléchargements, prouvant que le besoin était partagé par bien d'autres joueurs !

Architecture Mod & BepInEx

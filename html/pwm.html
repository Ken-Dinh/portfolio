Password Manager

7 Juillet 2023

Python Tkinter Cryptography

Application complète de gestion sécurisée des mots de passe.
Gestionnaire de mots de passe complet avec chiffrement, génération intelligente et stockage sécurisé. Premier grand projet Python combinant logique métier, interface graphique et sécurité informatique.

Architecture & Sécurité

Défis Techniques

✓ Intégrer chiffrement symétrique (Fernet)

✓ Gérer erreurs critiques (fichiers corrompus, clés incorrectes)

✓ Stockage sécurisé via keyring système

✓ Interface GUI stable avec Tkinter

✓ Générateur de mots de passe avancé

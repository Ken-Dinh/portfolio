Password Manager

7 Juillet 2023

Un projet complet autour de la sécurité et du chiffrement.
Lorsque j'ai commencé ce projet, je ne connaissais que les bases du Python et je n'avais jamais travaillé avec le chiffrement ni avec la création d'interfaces graphiques. L'objectif était simple : créer un gestionnaire de mots de passe. Mais il s'est rapidement transformé en une véritable initiation aux problématiques de sécurité.

Les principaux défis :
- Comprendre et intégrer le chiffrement symétrique (Fernet)
- Gérer les erreurs critiques : fichiers corrompus, clés incorrectes
- Stocker une clé de manière sécurisée via keyring
- Concevoir une interface stable avec Tkinter
- Créer un générateur de mots de passe avancé et personnalisable

Ce que j'ai appris :
- Les bases solides du chiffrement et du hachage (cryptography, hashlib)
- La gestion d'exceptions sensibles pour éviter les pertes de données
- L'organisation d'une application complète : UI, logique, sécurité, persistance
- Les bonnes pratiques essentielles pour manipuler des données sensibles

Ce projet m'a permis de passer de simples scripts Python à une approche plus structurée et professionnelle. Je comprends désormais beaucoup mieux les enjeux de sécurité et de robustesse dans une application manipulant des données critiques.

TXLForma

Projet Académique Collectif

Java Spring Boot MySQL Docker

Plateforme web complète de gestion de formations en numérique.
TXLForma est une application web permettant à une entreprise de formations de gérer ses cours, ses clients et ses ressources pédagogiques. Le projet inclut plusieurs interfaces pour les étudiants, professeurs et administrateurs, avec une API REST sécurisée et une architecture scalable.

Rôle dans le Projet

Mon Contribution

✓ Développement du backend avec Spring Boot

✓ Conception et gestion de la base de données MySQL

✓ Architecture et déploiement avec Docker

✓ Implémentation de l'API REST sécurisée

✓ Configuration du système d'authentification et autorisation

Structure du Projet

Équipe : 4 développeurs

Fonctionnalités : Gestion d'utilisateurs (Admin, Professeur, Étudiant), Catalogue de formations, Inscriptions, Gestion de sessions, Dashboard analytics

Compétences : Réseaux, DevOps, Design, Développement Front/Back, Cybersécurité, Conduite de projets

Objectif : Projet académique de fin d'année pour valider les compétences acquises en 5e semestre de licence informatique

Architecture API

L'API suit une architecture en couches avec sécurité par clé API, utilisant le pattern Controller → Service → Repository pour une séparation claire des responsabilités et une gestion centralisée des erreurs.
